# GLA是什么？

GLA，全称GL Addons，是由FPC5719编写的Windows平台上的OpeGL扩展库。其中对GL、GLU中的东西进行了整合，方便大家使用。

[在Github上访问](https://github.com/FPC5719/GLA)

# GLA怎么用？

我们来看这样一段代码：

```pascal
uses
	gla,windows;
begin
	glaInit;
	glaCreateWindow(100,100,640,480,'test');
	glaLoop;
end.
```

编译运行，我们会发现它在(100,100)的位置创建了一个640*480的窗口。
